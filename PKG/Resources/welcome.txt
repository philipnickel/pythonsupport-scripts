DTU Python Environment Installer

This installer will set up a complete Python development environment for DTU students, including:

• Python 3.11 via Miniconda
• Essential Python packages (pandas, scipy, statsmodels, etc.)
• DTU Math Tools package
• Visual Studio Code with Python extensions

The installation will automatically download and install the required components. An internet connection is required.

Installation typically takes 5-10 minutes depending on your internet connection.
